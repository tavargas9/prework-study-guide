HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The unordered list element creates a bullet point list.
  • The ordered list element creates a numbered list.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Link your style.css file using the link element as a child of the head element.
  • Always remeber the CSS Box Model when styling elements.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: pulls most recent main branch from GitHub repository.
  • git checkout main: switches current branch to main branch.
  • git branch -d branch-name: deletes branch locally.
  • git add -A: stages ALL files in the working branch for commit.
  • git commit -m "discriptive-message": commits the staged changes to your repository with a message attached to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a data structure that can store a collection of values within a variable.
  • A for loop is statement that allows you to execute a block of code repeatedly with conditions.
  • A function is a set of statements that performs a task or calculates a value.
